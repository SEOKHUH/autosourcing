## UX 개선

- **"크롤링 시작" 클릭 시 대기열 탭 자동 전환**
  후보에서 크롤링을 시작하면 자동으로 대기열 탭으로 전환해 진행 상황을 바로 볼 수 있게 하는 것.
  현재는 사용자가 직접 탭을 클릭해야 함.

---

## 기능 확장

- **쿠팡 소싱 오버레이 — 연관 검색어(검색태그) 제안**
  경쟁사 상품의 판매자 설정 검색태그는 외부에서 조회 불가.
  대신 쿠팡 검색 자동완성 API를 활용해 상품명 기반 연관 검색어를 가져와 태그로 제안하는 방식 검토.
  오버레이에 "추천 검색태그" 항목 추가 → 소싱 후보 데이터에 포함 → Step 4 검색태그 자동 입력까지 연결 가능.
