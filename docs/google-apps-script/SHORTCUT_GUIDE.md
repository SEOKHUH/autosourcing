# iOS 단축어 "쿠팡 소싱 추가" 셋업 가이드

## 준비물
- iPhone에서 **단축어** 앱 (기본 설치돼 있음)
- 위에서 복사한 webhook URL

---

## 단축어 만들기

### 1. 새 단축어 생성
- 단축어 앱 → 우측 상단 **+** 버튼
- 이름: "쿠팡 소싱 추가" (상단 제목 탭해서 수정)

---

### 2. 액션 추가 순서

아래 액션을 순서대로 추가합니다. 각 단계에서 **액션 추가(+)** 버튼을 눌러 검색.

---

#### 액션 1 — 공유 시트 입력 받기
- 검색: `단축어 입력`
- 액션명: **단축어 입력** (Shortcut Input)
- 설정: 없음 (기본값 유지)
- 역할: 공유 버튼으로 넘어온 URL을 받음

---

#### 액션 2 — URL 정규화 (m. → www.)
- 검색: `텍스트 바꾸기`
- 액션명: **텍스트 바꾸기** (Replace Text)
- 설정:
  - **입력**: `단축어 입력` (매직 변수 선택)
  - **찾기**: `m.coupang.com`
  - **바꾸기**: `www.coupang.com`
  - 대소문자 구분: **끄기**
- 결과를 `정규화된 URL` 변수에 저장 (액션 하단 "변수 추가" → 이름: 정규화된 URL)

---

#### 액션 3 — 딕셔너리 만들기
- 검색: `딕셔너리`
- 액션명: **딕셔너리** (Dictionary)
- 키-값 추가:
  - `url` : `정규화된 URL` (매직 변수)
  - `title` : `` (빈 문자열 — 나중에 페이지 제목 자동 추출 추가 가능)
- 결과를 `요청 본문` 변수에 저장

---

#### 액션 4 — JSON 변환
- 검색: `딕셔너리를 JSON으로`  
  또는 **텍스트** 액션에서 직접 JSON 작성 가능:
  ```
  {"url":"[정규화된 URL]","title":""}
  ```
  > **추천**: 아래 액션 5에서 딕셔너리를 직접 body로 사용하는 방식이 더 쉬움. 액션 3에서 만든 딕셔너리를 그대로 사용.

---

#### 액션 5 — POST 요청 보내기
- 검색: `URL 내용 가져오기`
- 액션명: **URL 내용 가져오기** (Get Contents of URL)
- 설정:
  - **URL**: 아래 URL 직접 입력
    ```
    https://script.google.com/macros/s/AKfycbxofCv4A2lA_ifdqSqCh8cNJnwag0DN1Haiz6i2ogmQ_4FJLrLfmY5oGzvst9Muxf4MQQ/exec?action=add
    ```
  - **메서드**: POST
  - **요청 본문**: JSON
  - **JSON 본문** → 키-값:
    - `url` : `정규화된 URL` (매직 변수)
    - `title` : `` (빈 문자열)
- 결과를 `서버 응답` 변수에 저장

---

#### 액션 6 — 알림 표시
- 검색: `알림`
- 액션명: **알림 표시** (Show Notification)
- 설정:
  - **제목**: `쿠팡 소싱`
  - **본문**: `위시리스트에 추가됨 ✅`

---

### 3. 공유 시트에서 실행되도록 설정

단축어 편집 화면 우측 상단 **⚙ (설정)** 버튼 → **공유 시트에서 보이기** 토글 **켜기**

---

## 사용 방법

1. Safari에서 쿠팡 상품 페이지 열기
2. 하단 **공유 버튼** (□↑) 탭
3. 공유 시트 아래로 스크롤 → **"쿠팡 소싱 추가"** 탭
4. 잠시 후 알림 "위시리스트에 추가됨 ✅" 표시

---

## 동작 확인

단축어 실행 후 구글 시트 `sourcing` 탭을 열어 새 행이 추가됐는지 확인.
