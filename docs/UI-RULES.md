# UI 디자인 시스템 규칙

> 헤오르 AutoSourcing 익스텐션 UI의 색상·타이포·컴포넌트 규칙.
> 모든 UI 작업 시 이 문서를 기준으로 적용.

---

## 색상 시스템

| 역할 | 색상 | 코드 |
|------|------|------|
| 배경 | 연회색 | `#f4f6fb` |
| 서피스 (카드·패널) | 흰색 | `#ffffff` |
| 테두리 | 연회색 | `#e2e4e9` |
| 텍스트 기본 | 다크 | `#1a1a2e` |
| 텍스트 보조 | 회색 | `#6b7280` |
| **액션 (Primary)** | **스카이블루** | **`#0ea5e9`** |
| 액션 Hover | 진한 블루 | `#0284c7` |
| 액션 연배경 | 연블루 | `#e0f2fe` |
| **에러 / 삭제** | **빨강** | **`#e94560`** |
| 성공 / 연결됨 | 그린 | `#16a34a` |

> 빨강(`#e94560`)은 삭제 버튼·에러 메시지 전용. 일반 액션에 절대 사용 금지.

---

## 타이포그래피

- **Font**: `'Inter'`, `'Noto Sans KR'`, `sans-serif`
- **Heading**: 15px / 700
- **Body**: 13px / 400
- **Label**: 12px / 600
- **Caption**: 11px / 400
- **줄간격**: 1.5

---

## 컴포넌트

### 버튼

| 종류 | 배경 | 텍스트 | 테두리 | 용도 |
|------|------|--------|--------|------|
| Primary | `#0ea5e9` | white | 없음 | 주요 액션 (크롤링 시작, 임시저장 등) |
| Ghost | white | `#555` | `1px solid #e2e4e9` | 보조 액션 (직접입력, 조회 등) |
| Danger | `#e94560` | white | 없음 | 삭제 전용 |

- radius: `8px`
- height: `36px` (기본) / `28px` (Small)
- font-size: `13px` (기본) / `12px` (Small)
- Hover: brightness 5% 어둡게

### 카드

- `background: #fff`
- `border: 1px solid #e2e4e9`
- `border-radius: 12px`
- `padding: 16px`

### 인풋

- `border: 1px solid #e2e4e9`
- `border-radius: 8px`
- `height: 36px`
- `padding: 8px 12px`
- `font-size: 13px`
- focus: `border-color: #0ea5e9`, `outline: none`

### 배지 / 태그

- `border-radius: 6px`
- `font-size: 11px`, `font-weight: 600`
- `padding: 2px 7px`

| 상태 | 배경 | 텍스트 |
|------|------|--------|
| 연결됨 | `#e0f2fe` | `#0284c7` |
| 대기중 | `#f0f2f8` | `#6b7280` |
| 에러 | `#fff0f2` | `#e94560` |
| 완료 | `#f0fdf4` | `#16a34a` |

### 탭

- 기본: `color: #6b7280`, `border-bottom: 2px solid transparent`
- 활성: `color: #0ea5e9`, `border-bottom: 2px solid #0ea5e9`
- 탭 영역: `border-bottom: 1px solid #e2e4e9`

---

## 레이아웃

- **헤더 없음**
- **기본 간격 단위**: 8px
- **섹션 간격**: 12px
- **카드 내부 여백**: 16px
- **사이드바 너비**: 300px (고정)
- **스플릿 레이아웃**: 좌(사이드바) + 우(워크스페이스) Flexbox

## 화면 구조 (탭 기반 사이드바)

```
┌──────────────────────────────────────────────────────┐
│  URL 입력창  [추가]                                    │
│──────────────────────────────────────────────────────│
│  [소싱 후보 N]  [대기열 N]   ← 탭                     │
│──────────────────────────────────────────────────────│
│  탭 콘텐츠 영역 (스크롤)                               │
└──────────────────────────────────────────────────────┘
```

워크스페이스 열리면 좌우 분할:
```
┌─────────────┬──────────────────────────────────────┐
│  사이드바    │  워크스페이스 (스텝 스테퍼 + 콘텐츠)  │
└─────────────┴──────────────────────────────────────┘
```
